HTML5 e CSS3

A linguagem HTML5 é focada em conteúdo, podendo ser: texto, imagens, vídeos, entre outros.

O CSS3 se foca em desing, como: Cores, sombras, tamanho, entre outros.

Também temos a linguagem de programação JavaScript, que trabalha com essas outras duas linguiagens(HTML5/CSS3), que já foca nas interações de um site.

Como fazer títulos e parágrafos?

Títulos

Para conseguir fazer um título basta utilizar no seu código a seguinte Tag: (< h1 > < /h1 >). Desse modo fazemos o título.

Parágrafos

Para conseguir fazer um parágrafo basta utilizar no seu código a seguinte Tag: (< p > < /p >). Desse modo fazemos o parágrafo.

Como colocar símbolos?

Para colocar símbolos especiais em seu site é bem simples, basta utilisar (&) acompanhado com o termo que puxa o símbolo, como: uarr, Delta, darr, entre outros, logo depois o (;).
